# Number Guessing Game

A fun number guessing game built using Python.

## Features
- Random number generation
- User input handling
- Hint messages (Too High / Too Low)
- Unlimited guessing attempts
- Win message when the correct number is guessed

## Technologies Used
- Python

## How to Run
1. Download or clone the repository.
2. Open the project in Python.
3. Run the program.
